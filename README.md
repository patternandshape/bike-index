#Bike Index

March 17, 2016

##### By Molly Curtin

### Description

An app that uses the Bike Index API to list all bikes that are stolen within a certain radius of the city entered by a user.


### Technologies Used

* JavaScript
* jQuery
* Gulp
* NPM
* Node
* AJAX
* Bike Index API
* HTML
* SCSS
